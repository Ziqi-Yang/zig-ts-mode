# Zig Tree Sitter Mode

Official repository is at CodeBerg:
[meow_king/zig-ts-mode](https://codeberg.org/meow_king/zig-ts-mode).

# Installation

## Elpaca

``` lisp
(use-package zig-ts-mode
  :ensure (:type git :host codeberg :repo "meow_king/zig-ts-mode"))
```

## Straight

``` lisp
(use-package zig-ts-mode
  :ensure '(:type git :host codeberg :repo "meow_king/zig-ts-mode"))
```
